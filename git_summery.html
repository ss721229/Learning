Git Summery

First : init, remote, status, add, commit

Second : config --global, log, push, clone

Third : branch, checkout, marge, reset, revert

    git init
    .git이라는 숨겨진 파일이 생성된다.
    git remote add origin (github url)
    로컬 저장소(PC)에 원격 저장소(github)를 연결하게 해준다.
    git status
    현재의 git 상태를 알려준다.
    변경된 사항이 있는 디렉터리를 알려준다.
    git add (Directory_Name)
    디렉터리를 스테이지 영역에 등록한다.
    디렉터리를 commit 하기 전 단계이다.
    git commit -m "Message"
    스테이지 영역에 있는 디렉터리를 commit한다.
    메시지는 로그를 확인할 때 볼 수 있다.
    git config --global user.name "User_Name"
    사용자의 이름을 git에 입력한다.
    git config --global user.email "User_Email"
    사용자의 이메일을 git에 입력한다.
    git log
    지금까지 이루어진 로그를 확인한다.
    git push -u origin (Branch_Name)
    commit이 완료된 디렉터리를 github에 저장한다.
    git clone (github url) .(현재폴더)
    init 명령어로 생성한 git저장소를 복제하여 사용한다.
    git branch (Branch_Name)
    새로운 branch를 생성한다.
    git branch -v
    branch 목록을 볼 수 있다.
    현재 선택한 branch는 *표시로 확인할 수 있다.
    git checkout (Branch_Name)
    입력한 branch의 작업 공간으로 이동할 수 있다.
    git marge (Branch_Name)
    현재 branch에 입력한 branch의 변경사항을 결합한다.
    git reset (위치)
    마지막 commit은 HEAD이며, 뒤에 ^, ^^를 붙이면 한 칸, 두 칸 이전 commit을 의미한다.
    commit 기록만 삭제되고, 작업 내용은 유지된다.
    git revert (위치)
    어떤 commit을 되돌리는 것인지 메시지를 채워준다.
