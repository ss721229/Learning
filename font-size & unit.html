font size

font-family

폰트 크기를 변경하려면 css의 font-size 속성을 사용하면 됩니다.

px 단위는 웹에서 사용되는 가장 기본적인 크기 단위입니다.

rem 단위는 html 태그의 폰트 크기를 기준으로 크기를 정하는 단위입니다.

em 단위는 부모 태그의 폰트 크기의 배수를 의미 합니다.

vw(viewport width)는 화면의 가로 크기와 관련 있는 단위입니다.
1vw는 화면 가로 크기의 1/100입니다.

vh(viewport height)는 화면의 세로 크기와 관련 있는 단위입니다.
1vh는 화면 세로 크기의 1/100입니다.
